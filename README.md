# Teoría de Coherencia Unificada (TCU)

**Proyecto de investigación empírica**

> Este repositorio contiene el desarrollo completo, reproducible y cronológico del proyecto TCU: un experimento orientado a evaluar la existencia de una **estructura matemática subyacente común** en las transiciones críticas de sistemas complejos adaptativos de múltiples dominios.

---

## 📂 Estructura del Repositorio

## Estado Actual

- **Hipótesis ontológica**: Formulada y refinada ✅
- **Validación empírica**: Completada en 10 dominios con datos reales públicos
- **Resultados principales**: 7 de 10 dominios con PASE claro (AUC promedio: 0.78)
- **Enfoque metodológico**: Modesto, reproducible, falsable y honesto con las limitaciones

---

## Principios del Proyecto

- Solo se trabaja con **datos reales** públicos.
- Nunca se generan datos sintéticos.
- Se mantiene trazabilidad cronológica y metodológica.
- La hipótesis se somete a criterios claros de falsación.
- El tono es académico, equilibrado y sin grandilocuencia.
